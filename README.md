## 설치
패키지 다운로드
```bash
git clone https://github.com/sesac-finance/Academinu_pakage.git
```

Academinu_package 폴더로 이동
```bash
cd Academinu_package
```

의존 라이브러리 설치
```bash
pip install -r requirements.txt
```
